User Manual

Data input

Three types of files need to be uploaded.

  • Raw data files: Single or multiple files can be uploaded
  • The plate layout file: One file that defines the plate layout.
  • The Treatment file: One file that defines the treatment.

Note: please download templates for these files and check that the column names of your files match the templates’ column names.

Previewing files in the main panel:

  • Raw files can be previewed (first three entries). A selection box is used for selecting different raw files if there are multiple raw files.
  • Treatment and layout can be viewed after both treatment and layout files have been uploaded. Users can view the plot of cells vs the selected value.

Outlier removal

Choose the focal raw file in the selection panel for the outlier removal process.

Users can set Robust Z-score, whether using pre-screen thresholds and pre-screen thresholds.

After clicking the <Remove outliers> button, outliers will be removed and plots will be displayed on the main panel. Plots can be viewed with the selected measurement (e.g. diameter, area, etc.).

Users can manually override cell results by selecting cells. The result of a selected cell will be toggled after applying the manual override (e.g. an outlier cell will be converted to a non-outlier cell).

The report of the selected file can be downloaded by clicking the <download> button.

Merging

A table of raw files will be shown on the main panel after uploading raw data files. The second column (column name: Processed) shows whether the file has gone through the outlier removal. The rest of the columns show the configuration (e.g. robust z score value) used in outlier removal.

Merged file and overall configuration are only available after all raw files have been processed.
