User Manual

Data input

Three types of files need to be uploaded.

  • Raw data file: Single or multiple files can be uploaded either taken directly from the output file from the confocal (the first template) or data can be manually inputted into the second template file. Any columns in which no data is present should have 0 for each row. 
  • The plate layout file: One file that defines the plate layout of treatments used. Numbers are defined by the treatment file.
  • The Treatment file: One file that defines the treatments. Multiple drugs and subsequent concentrations can be used.

Note: please download templates for these files and check that the column names of your files match the template's column names and sheet name matches the template's sheet name.

Previewing files in the main panel:

  • Raw files can be previewed. A selection box is used for selecting different raw files if there are multiple raw files.
  • Treatment and layout can be viewed after both treatment and layout files have been uploaded. Users can view the plot of cells vs the selected value.

Outlier removal

Choose the focal raw file in the selection panel for the outlier removal process.

Users can set Robust Z-score[1] and screen using pre-screen thresholds if required.

After clicking the Remove outliers button, outliers will be removed and plots will be displayed on the main panel. Plots can be viewed with the selected measurement (e.g. diameter, area, etc.).

Users can manually override cell results by selecting cells. The result of a selected cell will be toggled after applying the manual override (e.g. an outlier cell will be converted to a non-outlier cell).

The report of the selected file can be downloaded by clicking the download button.

Merging

A table of raw files will be shown on the main panel after uploading raw data files. The second column (column name: Processed) shows whether the file has gone through the outlier removal. The rest of the columns show the configuration (e.g. robust z score value) used in outlier removal. Downloaded outlier removal reports can also be uploaded and used in merging.

The merged file is only available when selected raw files have been processed.

Plotting

Plots of the merged data can be created and viewed. The app supports bar plots, point plots, dot plots and box plots.

Plots can be added to the report of files and downloaded.
